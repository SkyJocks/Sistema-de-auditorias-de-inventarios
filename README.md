Auditor de inventarios en python

Herramienta de automatización diseñada para procesar inventarios, filtrar productos disponibles y generar reportes de auditoría en formato CSV

Caracteristicas

Procesamiento Modular: toda la logica esta separada para calculos y la limpieza de datos
Auditoria Automatica: genera reportes en formato Exel(.csv)
Analisis de estadística: calcula automáticamente el promedio, precio maximo y minimo de los productos
Manejo de Rutas: se incluye manejo de rutas dinamicas y valida los datos vacios


## Instalación y uso ##
1. Clona el repositorio
2. Coloca tu archivo `productos.json` en la carpeta `data/`
3. Ejecuta: `python src/main.py`
